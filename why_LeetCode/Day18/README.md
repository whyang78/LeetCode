题目1：Linked List Cycle
  给定一个链表，判断链表中是否有环。

为了表示给定链表中的环，
  我们使用整数 pos 来表示链表尾连接到链表中的位置（索引从 0 开始）。 
  如果pos是 -1，则在该链表中没有环。




题目二：Linked List Cycle II
  给定一个链表，返回链表开始入环的第一个节点。如果链表无环，则返回null。
  

为了表示给定链表中的环，我们使用整数 pos 来表示链表尾连接到链表中的位置
 （索引从0开始。如果 pos 是 -1，则在该链表中没有环。
